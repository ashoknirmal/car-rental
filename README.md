# Car Connect

## Project info

Car Connect is a car rental booking application built with Vite, TypeScript, React, shadcn-ui, and Tailwind CSS.

## How can I edit this code?

To work on this project locally, clone the repository and install dependencies.

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Build the app with `npm run build` and deploy the generated `dist` folder to a static host or any compatible server.

## Can I connect a custom domain?

Yes, a custom domain can be configured through your hosting provider or deployment platform.
